### **Tools**

- [Beekeeper Studio](https://www.beekeeperstudio.io/)
  An intuitive, open-source SQL editor and database manager with a clean, modern UI.

- [pgAdmin](https://www.pgadmin.org/)  
  Official DB Management tool

- [DBeaver](https://dbeaver.io/)
  A powerful, universal database tool for developers and database administrators that supports any database with a JDBC driver.

- [dbdiagram.io](https://dbdiagram.io/home)
  A quick tool for designing database schemas and generating ER diagrams.
