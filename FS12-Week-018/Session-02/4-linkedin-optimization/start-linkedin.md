## Linkedin Profile Optimization

1. Create a linkedin account if you don't have one
2. Click on your profile name in top right from your home page
3. Add your shoulder up head shot to your profile
   - Use AI to help make it look more professional (Nano Banana)
   - Use keywords like (professional but not overly edited/AIed headshot)
4. Add Verification Badge
   - Multiple way to do this. One way is via work email
5. Edit Contact info
   - Add your portfolio, github
   - Don't add birthday or phone number
6. Hit edit icon on the top right (Edit Intro Modal)

- Add name
- Add a headline like (Software Engineer | Fullstack Developer | JavaScript, HTML, CSS, Express, MongoDB)
- Industry
- Education
- Location

7. Open to (button)

- Change types of work you are open to
- Put Recruiters only (you can make custom profile if you want your own circle)

8. Add Section > Add to Profile > Get Started > Next

- Select Resume > Browse Files
- Import the Exported PDF from OhMyCV
- Click Upload
- Work through Experience, Education, Skills Sections
  - remove or update any parts that are incorrect
  - if it fails mid-way go back to Add Section to continue
- Hit Review
- Uncheck Notify Network if you don't want to alert people in your network
- Add to Profile

9. Add Section (work through Core)

- Add about
- Services
- Any other sections you need to update

10. Add Section (work through Recommended)

- Add Featured
  - (Prior to this)Put a post up about a project (has an image/video)
  - Add that Post
  - Alternatively - link to your portfolio page for that project
- Add Licenses and Certifications
  - Add the bootcamp cert
- Add Project
  - Add projects here as well
- Add Courses
  - Add Courses from Codecademy
  - Add relevant courses from University
- Add Recommendations

11. Add Section (work through Additional)

- Optional

12. Start to post about your journey
 - Post about your final project and updates on it
 - Post about what you learn
 - Highlight positive/negative - people react emotionally - storytelling


 - Connecting with a lot of people will dilute your ability to get traction with posts
