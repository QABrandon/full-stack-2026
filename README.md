# full-stack-2026

Public portfolio for bootcamp and personal projects. The site **home page** is
**`index.html`** at the repo root (links into **`portfolio/`**). Each project lives under
**`portfolio/`** (e.g. `portfolio/personal-bio.html` plus its assets).

**Navigation:** Pages under `portfolio/` should link back to the hub with
`href="/"` (site root) when you add a new page or edit an existing one, so the
browser shows the domain root instead of `/index.html`. A redirect in
`vercel.json` also maps `/index.html` → `/`.
