# full-stack-2026

Public portfolio for bootcamp and personal projects. The **site home page** is
**`index.html`** at the repo root: the interactive responsive portfolio (projects,
skills, about, contact). Additional pages and assets live under **`portfolio/`**
(e.g. `portfolio/styled-landing-page.html`, `portfolio/personal-bio.html`).

**Navigation:** Pages under `portfolio/` link back to the site home with
`href="/"` so the address bar stays on the domain root. `vercel.json` maps
`/index.html` and `/portfolio/interactive-portfolio.html` → `/`.
