# full-stack-2026

Public portfolio for bootcamp and personal projects. The site **home page** is
**`index.html`** at the repo root (links into **`portfolio/`**). Each project lives under
**`portfolio/`** (e.g. `portfolio/personal-bio.html` plus its assets).

**Navigation:** Pages under `portfolio/` should link back to the hub home
(`../index.html` from a file in `portfolio/`) when you add a new page or edit
an existing one, so visitors can return to the project index.

Remote: [github.com/QABrandon/full-stack-2026](https://github.com/QABrandon/full-stack-2026).
