# Major 03: Personal Library App

FS12 Week 10 Session 3 — Major Project 3 (Option A). React + Vite library app with Google Books search, Context API, custom hooks, localStorage, favorites, filter/sort, and WCAG 2.1 AA patterns.

**Canonical source and class submission:** [github.com/QABrandon/personal-library](https://github.com/QABrandon/personal-library)

This folder is a portfolio pointer only. Clone the submission repo, then:

```bash
cd personal-library-app
npm install
npm run dev
npm run build
```

After deploy, add the live URL to the submission repo README and update the portfolio card link on the site home if needed.
