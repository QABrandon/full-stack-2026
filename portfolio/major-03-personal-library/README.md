# Major 03: Personal Library App

FS12 Week 10 Session 3 — Major Project 3 (Option A). React + Vite app for managing books with Google Books API, Context API, custom hooks, and localStorage.

**Canonical source lives here** in the `full-stack-2026` portfolio repo once Phase 10 sync completes.

## Features (planned)

- React Router — library, search, favorites, item detail
- Context API + custom hooks for library state
- Google Books API search
- localStorage persistence
- Favorites, reading status, filter/sort
- WCAG 2.1 AA accessibility

## Commands

```bash
cd personal-library-app
npm install
npm run dev
npm run build
```

After `npm run build`, open the portfolio card or visit `/portfolio/major-03-personal-library/dist/` on the deployed site.
