# Personal Library App

React + Vite front-end for the Personal Library major project.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Environment

Optional portfolio build base path:

```bash
VITE_BASE_PATH=/portfolio/major-03-personal-library/dist/ npm run build
```

## Stack

- React 19 + Vite
- React Router
- Context API
- Plain CSS design tokens
- Google Books API
- localStorage
