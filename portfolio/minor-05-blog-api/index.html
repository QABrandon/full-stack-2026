Blog Post API

A RESTful Express.js API for managing blog posts and users. Built with MVC routing, in-memory storage, validation, and centralized error handling. Tested with Postman and Newman.

Stack & architecture

  • Node.js
  • Express
  • REST
  • MVC
  • Postman

Routes live in routes/, business logic in controllers/, and shared error handling in middleware/. JSON docs are served at GET / when the server is running locally.

Posts — /api/posts

Method Route Action
GET /api/posts List all posts
GET /api/posts/:id Get one post
POST /api/posts Create post
PUT /api/posts/:id Update post
DELETE /api/posts/:id Delete post

Users — /api/users

Method Route Action
GET /api/users List all users
GET /api/users/:id Get one user
POST /api/users Create user
PUT /api/users/:id Update user
DELETE /api/users/:id Delete user
